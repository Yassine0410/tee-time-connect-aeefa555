# Tee Time Connect

## Clean project architecture

- frontend/: React + Vite application
- backend/: API service (to implement)
- database/: database infrastructure (Supabase config + SQL migrations)

## Run frontend locally

```sh
cd frontend
npm install
npm run dev
```

## Database

Supabase files are in `database/supabase`.
Migrations are in `database/supabase/migrations`.
